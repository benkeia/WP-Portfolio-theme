# Modèle 3D LEGO

Pour utiliser un modèle 3D LEGO personnalisé, placez votre fichier `LegoBrick.gltf` dans ce dossier.

Si aucun modèle n'est fourni, la scène utilisera automatiquement une géométrie de brique simple (BoxGeometry).

## Où trouver un modèle LEGO ?

- [Sketchfab](https://sketchfab.com/search?q=lego+brick&type=models) - Recherchez "lego brick" et téléchargez un modèle gratuit
- [Free3D](https://free3d.com/) - Modèles 3D gratuits
- Ou créez le vôtre avec Blender

Assurez-vous que le fichier est au format GLTF (.gltf) et nommez-le `LegoBrick.gltf`.
