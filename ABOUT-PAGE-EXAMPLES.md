## 💡 Conseils de Rédaction

### Pour la Biographie
- **Soyez authentique** : Écrivez à la première personne, montrez votre personnalité
- **Restez concis** : 2-3 paragraphes maximum, allez à l'essentiel
- **Mettez en avant vos forces** : Qu'est-ce qui vous rend unique ?
- **Incluez un CTA subtil** : Invitez les visiteurs à explorer davantage

### Pour les Compétences
- **Organisez logiquement** : Groupez par catégories cohérentes
- **Priorisez** : Mettez vos compétences principales en premier
- **Soyez honnête** : N'ajoutez que les compétences que vous maîtrisez vraiment
- **Restez à jour** : Mettez régulièrement vos compétences à jour

### Pour le Parcours
- **Ordre chronologique inversé** : Le plus récent en premier
- **Soyez précis** : Indiquez les dates et entreprises
- **Quantifiez** : Mentionnez les résultats concrets si possible
- **Évitez le jargon** : Restez accessible

### Pour les Valeurs
- **Maximum 4-6 valeurs** : Trop de valeurs dilue le message
- **Soyez spécifique** : Expliquez concrètement ce que chaque valeur signifie pour vous
- **Illustrez** : Donnez des exemples si possible
- **Restez sincère** : Ne copiez pas des valeurs génériques

## 📸 Recommandations pour l'Image Hero

### Caractéristiques techniques
- **Dimensions** : 1920 x 1080 pixels minimum
- **Format** : JPG ou WebP
- **Poids** : < 500 KB (optimisé)
- **Orientation** : Paysage

### Suggestions de contenu
1. **Portrait professionnel** sur fond neutre ou artistique
2. **Espace de travail** : Votre setup, votre bureau
3. **Photo artistique** : Image abstraite ou créative qui vous représente
4. **Gradient coloré** : Alternative simple et moderne
5. **Photo d'action** : Vous en train de coder, lors d'une conférence, etc.

### Où trouver des images
- **Photos personnelles** : Le plus authentique !
- **Unsplash** : Photos libres de droits haute qualité
- **Pexels** : Grande variété d'images gratuites
- **Pixabay** : Banque d'images libres

### Outils d'optimisation
- **TinyPNG** : Compression sans perte de qualité
- **Squoosh** : Convertisseur WebP
- **Remove.bg** : Supprimer l'arrière-plan
